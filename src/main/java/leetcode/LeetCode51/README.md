# LeetCode 51 - N-Queens

## Problem
Place `n` queens on an `n x n` board so that no two queens attack each other, and return all valid board configurations.

## Implementation
- Source: `Solution.java`
- Technique: backtracking with validity checks (row and diagonals)

## Example
- Input: `n = 4`
- Output: all valid 4-queen board layouts
