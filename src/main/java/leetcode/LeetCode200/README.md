# LeetCode 200 - Number of Islands

## Problem
Given a `2D` grid of `1`s (land) and `0`s (water), count how many islands exist.

## Implementation
- Source: `Solution.java`
- Technique: DFS flood-fill (mark visited land as water)

## Complexity
- Time: `O(rows * cols)`
- Space: recursion stack in worst case
